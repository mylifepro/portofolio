# RAMARISANTOANINA Rochel - Portfolio

![Portfolio Preview]
Un portfolio moderne développé avec **React + Tailwind CSS**, mettant en avant mes projets, compétences et parcours.

## 🌟 Aperçu

[Visiter le site en ligne →] 


## ✨ Fonctionnalités

- Design moderne et responsive
- Animations fluides (Framer Motion)
- Mode sombre / clair
- Navigation fluide
- Section Projets avec filtres
- Formulaire de contact fonctionnel
- Optimisé pour le SEO et les performances

## 🛠 Technologies utilisées

- **Frontend** : React.js, Tailwind CSS
- **Animations** : Framer Motion
- **Icônes** : React Icons / Lucide
- **Déploiement** : Vercel / Netlify
- **Autres** : React Router, Axios, etc.

```bash
